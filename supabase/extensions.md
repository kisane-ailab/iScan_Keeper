# 설치된 PostgreSQL 확장

## 활성화된 확장

| 확장명 | 스키마 | 버전 | 설명 |
|--------|--------|------|------|
| pgcrypto | extensions | 1.3 | 암호화 함수 |
| pg_stat_statements | extensions | 1.11 | SQL 통계 추적 |
| supabase_vault | vault | 0.3.1 | Supabase Vault 확장 |
| pg_net | extensions | 0.19.5 | Async HTTP |
| pg_graphql | graphql | 1.5.11 | GraphQL 지원 |
| uuid-ossp | extensions | 1.1 | UUID 생성 |
| pg_cron | pg_catalog | 1.6.4 | 작업 스케줄러 |
| plpgsql | pg_catalog | 1.0 | PL/pgSQL 언어 |
